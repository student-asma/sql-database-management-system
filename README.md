# SQL Database Management System

## 📌 Project Overview

This project was completed as **Task 3** during my **Data Analytics Internship at DecodeLabs**.

The objective was to design and manage a relational database using MySQL Workbench while applying core SQL concepts such as table creation, relationships, CRUD operations, joins, views, indexes, and ER diagram design.

---

## 🛠️ Tools & Technologies

- MySQL Workbench
- SQL
- Database Management System (DBMS)

---

## 📂 Database Structure

The database includes the following tables:

- Employees
- Departments
- Projects
- EmployeeProject
- Salaries

Additionally:

- Employee_Department_View
